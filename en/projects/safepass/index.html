SafePass

Let's you create a strong password by entering a message and a key

Safe passwords for everyone!

A safe password is crucial these days, but using safe passwords is often too complicated.
With SafePass, this becomes a thing of the past.
The idea is simple: You enter a message and a key, which generates a password using these inputs.
The usage can be customized to your needs. Use the exact same message every time with a different key? No problem.
Enter the identical key with a different message? We got you covered.
And the best thing is: It's open-source. You can download and inspect the code or change it to your needs.
